Brendan Eich

How he found the idea.

Millions of developers use a programming language today that was created in just 10 days during the hustle and bustle of the dot-com boom. JavaScript creator Brendan Eich revisited the roots of his language in some newly-released online videos and explained how all the seeds he planted in 1995 are finally coming to fruition. 1995 was a different era and a different world. Browsers had the potential to let developers reach users on any platform — windows, Mac, or Linux — but the missing ingredient was a programming language. “I was invited to join at the beginning, which I did not because I was sticking with a previous startup for another year,” Eich recently told InfoWorld. “Back then, the pace of Web innovation was furious, with Microsoft suddenly making the Internet the focus of its Windows 95 operating system release in response to Netscape's emerging browser and server products.” So famously, 34-year-old Brendan Eich created the first version of the language that would become JavaScript in just 10 days. We do not know if we even need a little language like we wanted you to do. Maybe Java is enough.' “It was also an incredible rush job, so there were mistakes in it. I think it is important because I knew there would be mistakes, and there would be gaps, so I made it very malleable as a language.” After 23 years of reflection, is there anything had to do differently? People tell him he should’ve refused to work on such a short schedule — or that he should have implemented a different language into Netscape, like Perl or Python or Scheme — but that is not what he would have changed. He wishes he’d been more selective about which feedback he’d listened to from JavaScript’s first in-house testers. though Eich jokes at the end of the interview that if you trace it all the back to its origins, JavaScript is one of the fundamental pieces that advertisers use for web tracking. “So I'm making up for it,” he says with a laugh.

Who is he.

Brendan Eich is an American technologist and creator of the JavaScript programming language. He founded the Mozilla project and Mozilla Corporation , and served as CTO and briefly CEO of Mozilla Corporation.He is the CEO of Brave Browser .
